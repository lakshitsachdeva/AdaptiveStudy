Cognitive Load-Aware Interface Design

AdaptiveStudy — HCI Prototype Research Report

Behavioral signal fusion for real-time interface adaptation in educational environments

Abstract

AdaptiveStudy demonstrates real-time cognitive load estimation through passive behavioral monitoring, adapting the study interface to reduce extraneous cognitive load and support learning efficiency.

Methodology
  • Cursor Entropy (25%): Angular deviation in movement vectors
  • Hesitation Index (35%): Inter-event timing and hover pauses
  • Error Rate (25%): Delete/backspace bursts in a 30s window
  • Scroll Rhythm (15%): Speed and direction reversals in 10s
  • EMA Smoothing: α = 0.3 with debounced mode switching
HCI Principles Applied
  • Cognitive Load Theory (Sweller, 1988)
  • Miller's Law through chunk reduction in Calm Mode
  • Fitts' Law with larger relaxed targets under load
  • Progressive Disclosure as complexity reduction
  • Aesthetic-Usability Effect through calm visual design
  • Nielsen Heuristic #1 through visible feedback and state toasts

Limitations
  • Behavioral proxies, not physiological ground truth
  • No EEG, pupillometry, or GSR integration
  • Rule-based thresholds rather than learned models
  • Static deck rather than cloud-backed content
Future Work
